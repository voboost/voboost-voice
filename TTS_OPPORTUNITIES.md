## 🚀 РЕКОМЕНДУЕМЫЕ ДЛЯ РЕАЛИЗАЦИИ

### Приоритет 1 (самые полезные):
1. ✅ **Погода** (ACTION_WEATHER_QUERY) — уже есть в Ivoka
2. ✅ **Навигация** (ACTION_DCS_POI_NAV, ACTION_DCS_ROUTELINE_NAV) — уже есть в Ivoka
3. ✅ **Телефон** (ACTION_TELEPHONE_VIEW, ACTION_VOICE_PHONE_CALLBACK) — частично реализовано
4. ✅ **Музыка** (ACTION_MUSIC_PLAY, ACTION_CMD_PLAY/PAUSE/NEXT/PREV) — не реализовано
5. ✅ **Такси/Доставка** (ACTION_VOICE_CALL_TAXI_QUERY, ACTION_VOICE_EXPRESS) — не реализовано

### Приоритет 2 (удобные для реализации):
6. ✅ **Радио** (ACTION_RADIO_LAUNCH, ACTION_CMD_PLAY) — не реализовано
7. ✅ **Будильник** (ACTION_DCS_COMMON_ALARM_QUERY) — не реализовано
8. ✅ **Новости** (ACTION_NEWS_PLAY) — не реализовано
9. ✅ **Яркость** (ACTION_DCS_CAR_SET_BRIGHTNESS) — не реализовано
10. ✅ **Парковка** (ACTION_DCS_PARKING_NAV, ACTION_DCS_PARKING_PAY) — не реализовано

### Приоритет 3 (дополнительные):
11. Отключить IOT (мало relevant для автомобиля)
12. Отключить_COFFEE (не automotive-specific)
13. Реализовать только если будет запрос

## 📋 ГЛАВНЫЕ ВЫВОДЫ

### Что уже есть в Voboost:
- ✅ Климат (AIR_CONTROL) — **100%**
- ✅ Лючок/бензобак (CHARGPORT/FUELTANK) — **100%**
- ✅ Режимы (SMART_MODE) — **100%**
- ✅ Окна (WINDOW) — **100%**
- ✅ Телефон (TELEPHONE_CALL) — **частично (только по номеру)**

### Что можно добавить:

## 🔍 МЕТОДОЛОГИЯ АНАЛИЗА

### Что анализировалось:
1. **VoiceActionID.java** — 486 ACTION_ID для QGTtsService
2. **TtsService.java** — как TTS передаётся через `speak()` и `playAudio()`
3. **VuiTtsObj.java** — параметры TTS: `speakContent`, `displayText`, `mood`, `location`, `permission`
4. **AbstractTtsImpl.java** — интерфейс `speak(String text)` и `TtsCallback` (onDone, onError, onStart)

### Как использовать TTS в Voboost:
```kotlin
// В VoboostVoiceService
ttsEngine.speak("Привет! Погода сегодня +22°C, ясно")

// С параметрами (через Bundle)
val params = Bundle().apply {
    putString("params_emotion_level", "cheerful")  // настроение
    putString("play_location", "front_right")       // зона
}
ttsEngine.speak("Мы прибыли на место назначения", params)
```
