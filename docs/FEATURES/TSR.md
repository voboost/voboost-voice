# TSR (Система предупреждения о превышении скорости)
**Статус:** ✅ Реализовано (частично)
**Описание:** Система предупреждает водителя о превышении установленного лимита скорости через ISA (Intelligent Speed Assistance)
Голосовой ассистент интегрирован с системой ограничения скорости (ISA) автомобиля. При превышении заданного лимита озвучивается предупреждение.

**Важно:** В текущей версии реализовано только **предупреждение о превышении**, без детекции дорожных знаков (TSR_SIGN_TYPE).
## 📡 Источники данных ISA/TSR
| `ISA_ISLC_STATUS` | CAN-битовая маска | ⭐⭐⭐⭐⭐ | Статус превышения скорости (7 = превышен) |
| `ISA_OVER_SPEED_WARNING_SWITCH` | CAN-битовая маска | ⭐⭐⭐⭐ | Включен/выключен режим ISA |
| `VehicleState.speed` | CAN-значение | ⭐⭐⭐⭐⭐ | Текущая скорость автомобиля |
| GPS + OSM/HERE | Оффлайн-база | ⭐⭐⭐⭐ | Резервный источник лимитов (не реализован) |
### ISA_ISLC_STATUS
Статус системы ограничения скорости.

| Код | Значение | Действие |
|-----|----------|----------|
| `7` | **Превышение скорости** | Озвучить предупреждение |
| `2-6` | Другой статус | Игнорировать |

### ISA_OVER_SPEED_WARNING_SWITCH
Переключатель режима ISA.
| `2` | **Включено** |
| `1` | Выключено |

## ⚠️ Известные проблемы

| Проблема | Причина | Решение |
|----------|---------|---------|
| TSR не срабатывает | Камера забита грязью/снегом | Проверить статус `TSR_OPERATING_STATUS` |
| Ложные срабатывания | Низкое качество камеры ADAS | Использовать GPS+OSM как резерв |
| Задержка озвучивания | Высокая нагрузка на процессор | Оптимизировать поток данных |
