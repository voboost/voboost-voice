# Voboost Voice

**English** | [Русский](README.ru.md)

On-board voice assistant for Voboost vehicles. Recognizes spoken commands and
dispatches them to vehicle systems (climate, windows, driving modes, etc.) via
the QGBus and canbus AIDL services.
