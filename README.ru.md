# Voboost Voice

[English](README.md) | **Русский**

Бортовой голосовой ассистент для автомобилей Voboost. Распознаёт голосовые
команды и передаёт их в системы автомобиля (климат, окна, режимы движения и
т.д.) через сервисы QGBus и canbus AIDL.
